# Catálogo Capuchones

Sistema web para control de inventario interno de capuchones de flores.
Permite gestionar y visualizar los productos disponibles de forma sencilla.

## Tecnologías
- HTML
- CSS
- JavaScript

## Funcionalidades
- Visualización de productos
- Control de inventario interno
- Organización por categorías

## Cómo verlo
Clona el repositorio y abre `index.html` en tu navegador.
```bash
git clone https://github.com/alexismoreno1211-tech/Catalogo-capuchones.git
```
